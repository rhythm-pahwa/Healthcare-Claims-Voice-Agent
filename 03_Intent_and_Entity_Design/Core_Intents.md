# Core Intents

After analyzing the healthcare claims use cases, we identified the following core intents that the voice agent should support.

## Claim Status

Allows members or providers to check the current status of a submitted claim.

## Claim Submission

Allows users to submit claim related information for processing.

## Eligibility Check

Allows users to verify healthcare coverage eligibility.

## Benefits Inquiry

Allows users to obtain information regarding healthcare benefits and coverage.

## Provider Lookup

Allows users to search for healthcare providers within the network.

## Pre Authorization Status

Allows users to check the status of pre authorization requests.

## Service Request

Allows users to create service requests for issues requiring additional assistance.

## Agent Escalation

Allows users to connect with a human representative when necessary.
